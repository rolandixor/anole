anole docs - text formatting
.an-tips
Easily add a tip to element using
.an-tip
.
.an-tip-parent
is required on the element that contains the tip. You can also use
.an-text-tip
if your tip is attached to a block (or span) of text.

See the example below:
<div class="">Hey kid, I've got a tip:
    <div class="an-text-tip an-tip-parent">I'm a tip, homie.</div> hover/tap the underlined text.
</div>
