anole framework demo
Welcome to the Anole Demo!
This page gives a demonstration of the features and powers of Anole. It contains almost all of the components and animations of the framework. Anything that isn't included here will be covered in the documentation. By default, Anole uses the Lato font family, but you can override this with your own css. Also, Anole is themed out the box, so you can drop it in your html and go.

If you would like to try other themes, just swap out the existing css for another theme and see how it looks!.

Place your mouse over any ".an-reveal" to see its corresponding action. Anole reveal animations can be triggered by a simple hover (or tap on mobile), or you can trigger them programatically by adding the "active" class via Javascript. Since these classes use transitions to produce their animations, you can easily adjust their timing by overriding the corresponding
